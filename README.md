# Ramadan Helper - Islamic Web Application

A comprehensive Islamic web application providing:
-  AI-powered Islamic Q&A with Quran & Hadith references
-  Personalized Dua Generator
-  Islamic Video Search
-  Chat with Real Imams

## Features
- Ask AI questions and get answers with relevant Quran ayahs and Hadiths
- Generate personalized duas based on your situation
- Search for Islamic educational videos
- Have private conversations with qualified Imams

## Tech Stack
- **Frontend**: HTML, Tailwind CSS, JavaScript
- **Backend**: FastAPI (Python)
- **Database**: SQLite
- **AI**: Groq API
